Working with Image/Picture

If an article already has an infobox at the top right, then the usual place for the article's first picture is within the infobox. For guidance on the syntax for doing this, see Help:Infobox picture. In very brief summary, one hurdle that trips up many people when attempting to add an image to an infobox template is that most internally provide the wiki code that "wraps" the image. Accordingly, you do not usually add the brackets, number of pixels, and other code details you will learn about below, when placing an image in infoboxes â€“ just the file name next to a field labeled | image =. Adding such extraneous code will cause many infoboxes to break. Also, be aware that some infoboxes require that the file's name be placed without the file/image namespace prefix. Thus, and for example, if File:Name.jpg does not work, try just Name.jpg.
